Wynnbot, a Python project of mine

A utility discord bot for the minecraft server Wynncraft, with features such as

  • Listing the number of players online in each world
  • Listing the usernames of each player in a specified world
  • Searching for ingredients by name
  • Displaying the total level leaderboard
  • Displaying the stats of a specified user's classes
  • Showing the 10 most inactive players in the guild I'm in
  • Checking how many people are online in the guild I'm in
  • Checking what guild a specified player is in
