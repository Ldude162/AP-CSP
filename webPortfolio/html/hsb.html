Hypixel Skyblock

Hypixel Skyblock is a minigame on the Hypixel Minecraft Server. It requires a lot of "grinding", which is doing a lot of boring tasks over and over until you get better items. There are a lot of cool mechanics in Hypixel Skyblock, like the item ability system which allow you to use cool abilities on items.
